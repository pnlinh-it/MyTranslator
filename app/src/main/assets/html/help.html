1. Using Keyboard Screen

  • 1Input text to translate
  • 2Tap the "Translate" button to translate the input text

  • 3Show translation results
  • 4Change source language
  • 5Change target language
  • 6Switch source and target language
  • 7Tap to show the keyboard
  • 8Tap to go to the voice screen
  • 9Tap to go to the camera screen
  • 10Tap to show the setting menu

2. Using Voice Screen

  • 1Tap to input your speech
  • 2Show recognized text from your speech
  • 3Change source language
  • 4Change target language
  • 5Switch source and target language
  • 6Tap to go to the keyboard screen
  • 7Tap to go to the camera screen
  • 8Tap to show the setting menu

  • 9Show translated text

3. Using Camera Screen

  • 1Tap to take a photo
  • 2Show translated text from your photo
  • 3Change source language
  • 4Change target language
  • 5Switch source and target language
  • 6Tap to go to the keyboard screen
  • 7Tap to go to the voice screen
  • 8Tap to show the setting menu

4. Using Setting Screen

  • 1Tap to choose one item for the first start screen
  • 2Tap to clear all history data

5. Using History Screen

  • 1Tap and hold to open the deleting mode

  • 2Check or uncheck to select or unselect the item
  • 3Tap to delete all selected item
  • 4Select or unselect all item
  • 5Go back to the previous screen
